Projects

Srvcast: Stateful and Scalable Anycast System for Computing-Aware Networks
Srvcast is a network-layer system designed for computation-aware networks. It enables stateful and scalable service-oriented anycast communication by integrating anycast routing, segment routing, and programmable data-plane mechanisms.

The system introduces:
  • Two-stage routing for stateful SID task steering
  • ServiceNAT for SID-to-host transparent socket compatibility
  • Selective SID advertisement for scalable routing
  • P4-based programmable data-plane implementation
